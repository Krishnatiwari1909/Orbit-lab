# OrbitLab 🛰️
**AI-Driven Virtual Satellite Engineering Platform**

*Official Submission for India Innovation 2026*

## ⚠️ The Problem
Satellite engineering tools are highly complex, overwhelmingly expensive, and require an advanced understanding of calculus and orbital mechanics. This creates a massive barrier to entry for most STEM students and colleges, making hands-on aerospace education largely inaccessible.

## 💡 The Solution
OrbitLab is a web-based, AI-powered aerospace simulator. It utilizes a Generative AI Copilot to translate natural language into complex orbital physics, empowering users to intuitively design, simulate, and operate CubeSats directly in their browser without needing a PhD in mathematics.

## ✨ Key Features
* **Generative AI Mission Copilot:** Instantly calculates deterministic orbital physics (altitude, velocity, mass, inclination) from simple user prompts using the Gemini API.
* **Cinematic 3D Simulation:** High-fidelity, hardware-accelerated Earth rendering utilizing advanced Quaternion SLERP interpolation for flawless, mathematically accurate global targeting and tracking.
* **Virtual Satellite Assembly:** Component-level configuration for custom CubeSats (Payloads, Solar Wings, OBC, S-Band Radios).
* **Live Telemetry:** Real-time simulation dashboard displaying power consumption, solar input, and active orbital link status.

## 🎯 Target Users
* College engineering and STEM students
* Universities and educational coding bootcamps
* Space enthusiasts and early-stage aerospace startups

## 🗺️ Roadmap
* **Phase 1: AI-Powered Web Simulation (Current)** - Launching the core 3D platform and Gemini Copilot integration for India Innovation 2026.
* **Phase 2: B2B EdTech SaaS** - Licensing the platform to educational institutions for interactive classroom use.
* **Phase 3: Edge AI Integration** - Transitioning to local, open-source AI models using on-device NPUs for offline simulations, leading directly into physical CubeSat prototyping.

## 💻 Tech Stack
* **Frontend:** HTML/CSS/JS, Three.js (WebGL optimized for client-side GPUs)
* **Backend:** Node.js, Express.js (Secure API routing and CORS handling)
* **AI Engine:** Google Gemini Generative AI 
* **Cloud Infrastructure:** Designed for scalable, full-stack deployment via Azure Static Web Apps and serverless functions.
