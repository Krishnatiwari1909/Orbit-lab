# OrbitLab – Virtual Satellite Engineering Platform

## Problem
Satellite engineering tools are expensive, complex, and inaccessible to most students and colleges in India.

## Solution
OrbitLab is a software-first virtual lab that allows users to build, simulate, and operate a CubeSat using only a browser.

## Features
- Virtual satellite assembly
- Real-time power and mission simulation
- Component-level understanding
- Runs on low-end hardware

## Target Users
- Engineering students
- Universities
- Early-stage space startups

## Roadmap
Phase 1: Software simulation (current)  
Phase 2: CubeSat prototype using this design  
Phase 3: Micro-satellite hardware company

## Tech
- Three.js
- Web-based simulation
- No hardware required
